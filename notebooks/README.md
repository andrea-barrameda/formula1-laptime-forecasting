# Notebooks

This folder contains all Jupyter notebooks used throughout the project.

## Planned Workflow

| Notebook | Purpose |
|----------|---------|
| 01_data_collection.ipynb | Download and organize Formula 1 data |
| 02_data_cleaning.ipynb | Clean and preprocess raw data |
| 03_exploratory_analysis.ipynb | Explore patterns and relationships |
| 04_feature_engineering.ipynb | Create predictive features |
| 05_baseline_models.ipynb | Develop statistical baseline models |
| 06_time_series_models.ipynb | Train ARIMA and other forecasting models |
| 07_machine_learning_models.ipynb | Train Random Forest, XGBoost, etc. |
| 08_model_interpretation.ipynb | Explain model predictions |
| 09_final_evaluation.ipynb | Compare models and summarize results |
